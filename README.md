# Unit Converter · converter.lab115.com

Free online unit converter with 8 categories: Length, Weight, Temperature, Volume, Area, Speed, Time, Data.

## Features
- Instant conversion as you type
- 50+ units across 8 categories
- Quick reference table
- Copy result to clipboard
- Privacy-first: all conversions in-browser
- Google Analytics + AdSense ready

## Tech
- HTML/CSS/JS (vanilla)
- Nginx (Docker)
- Multi-arch GitHub Actions build (arm64 + amd64)
- Coolify auto-deploy
